# 🎬 Movie Recommendation Dashboard (Power BI)

## 📌 Overview

This project is a **Movie Recommendation Dashboard built using Power BI**, designed to analyze movie data and provide insights that help users discover movies based on **genres, ratings, and popularity trends**.

The dashboard allows users to interact with the data and explore patterns that can support recommendation decisions.

## 🎯 Objectives

* Analyze movie dataset to uncover meaningful insights
* Build an interactive dashboard for better visualization
* Enable users to explore movies based on filters like genre and year
* Demonstrate data analysis and visualization skills

## 🛠️ Tools & Technologies

* **Power BI** – Dashboard creation
* **Power Query** – Data cleaning & transformation
* **Excel / CSV Dataset** – Data source
* **DBbrowser** - To clean & filter data

## 📂 Dataset

The dataset contains the following features:

* Movie Titles
* Genres
* Ratings
* Release Year
* Popularity / Votes

This data is used to analyze trends and support movie recommendations.

## ⚙️ Workflow

### 1. Data Cleaning

* Removed null and missing values
* Eliminated duplicate records
* Selected relevant columns for analysis

### 2. Data Transformation

* Created calculated columns and measures
* Organized genre and rating data
* Structured dataset for visualization

### 3. Dashboard Development

* Built interactive visuals using Power BI
* Added filters (slicers) for better user experience
* Designed layout for clear insights


## 📊 Dashboard Features

* 🎯 Genre-based filtering
* 📊 Rating distribution visualization
* 📈 Trends across years
* ⭐ Top-rated and popular movies


## 💡 Key Insights

* Certain genres consistently receive higher ratings
* Popularity trends vary across different years
* A small number of movies dominate high ratings
* Filters help in exploring personalized preferences
